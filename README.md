## About The Project

This is an online learning platform in the same vein as Udemy and other MOOCs. You have a section for teachers and another section for students

Students can:

- Buy courses
- Explore courses
- Simulate taking classes

Teachers can

- Create courses
- Explore courses
- Edit Courses
